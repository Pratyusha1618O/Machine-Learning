Mall Customer Segmentation

Download the customer dataset from the following link:

* **Data Set:** [Kaggle - Mall Customers Dataset](https://www.kaggle.com/shwetabh123/mall-customers)

### Overview
This dataset gives the data of income and money spent by customers visiting a shopping mall. The dataset contains **Customer ID**, **Gender**, **Age**, **Annual Income**, and **Spending Score**. 

As a mall owner, you need to identify the group of people who are the most profitable customers. Apply **at least two clustering algorithms** (based on Spending Score) to group the customers.

### Tasks
- **a.** Apply Data Pre-processing (Label Encoding, Data Transformation, etc.) techniques if necessary.
- **b.** Perform Data Preparation (Train-Test Split).
- **c.** Apply Machine Learning Algorithms.
- **d.** Evaluate Model.
- **e.** Apply Cross-Validation and Evaluate Model.
